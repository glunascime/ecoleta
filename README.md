O que é?
- Uma plataforma online criada para facilitar a conexão entre estabelecimentos de coleta de resíduos recicláveis/orgânicos e pessoas em busca do descarte adequado com eficiência.

 Tecnologias utilizadas:
- [Node.js](https://nodejs.org/en/)
- [Express](https://expressjs.com/pt-br/)
- [SQLite](https://www.sqlite.org/index.html)
- [Nunjucks](https://mozilla.github.io/nunjucks/)

Layout:
- A partir [desse link](https://www.figma.com/file/Byw4X5etg8VCmezueyhzkC/Ecoleta-(Starter)?node-id=136%3A546) você pode visualizar o layout do projeto. Mas lembra: precisa estar logado na sua conta Figma!

Execução:
- Clone o repositório (``` git clone https://github.com/glunascime/ecoleta.git``` )
- Entrar no diretório (``` cd ecoleta``` )
- Baixar as dependências (``` npm i``` )
- Executar o servidor (``` npm run -> nodemon src/server.js```)

Apareceu ```starting node src/server.js```? Pronto! 

Só acessar 'http://localhost:3000' no seu navegador e aproveitar.
